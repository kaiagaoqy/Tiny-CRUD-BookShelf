# Tiny-CRUD-BookShelf
## Stack
1. Framework: Node.js, express, ejs
2. DataBase: MongoDB


## Deployment
 "dependencies": {
    "body-parser": "^1.20.0",    
    "ejs": "^3.1.8",    
    "express": "^4.18.1",    
    "lodash": "^4.17.21",    
    "mongoose": "^6.6.1",    
    "nodemon": "^2.0.20"    
  }    

```js 
npm i body-parser
npm i express   
npm i mongoose 
npm i lodash
npm i ejs
```    
## Run
``` js
node main.js
```
